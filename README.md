# Water-Meter-Reading-App
An intelligent computer vision utility optimized for utility data management. This system extracts consumption indices from analog water meters by dynamically segmenting physical frameworks (meters and glass windows) from consumption numerical digits.   Equipped with a custom inference layout, the pipeline handles challenging field-camera tilts, severe light-glare reflections, and contains a specialized fallback remapping mechanism that isolates unidentifiable anomalies into 'unknown' tokens rather than generating false digits, keeping accuracy scores reliably high.
